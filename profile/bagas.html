What do I learn?

  1. Melalui mata kuliah Dasar-Dasar Komputasi, saya memperoleh pemahaman yang baik tentang Microsoft Excel, khususnya dalam penggunaan fungsi-fungsi seperti SUM, AVERAGE, IF, dan VLOOKUP. Saya juga belajar membuat tabel, grafik, dan memanfaatkan fitur filter serta pemformatan data untuk menyajikan informasi secara rapi dan terstruktur, yang sangat berguna dalam pengolahan data.
  2. Pada materi Microsoft Word, PowerPoint, dan Canva, saya dilatih membuat dokumen dan presentasi yang efektif serta menarik. Penggunaan heading, daftar isi otomatis, dan pengaturan format di Word sangat membantu dalam penulisan akademik. Sementara itu, kemampuan menyusun presentasi dengan desain visual yang komunikatif di PowerPoint dan Canva mendukung keterampilan komunikasi saya.
  3. Pembelajaran HTML dan CSS memberikan dasar penting dalam memahami struktur dan tampilan halaman web. Saya belajar menulis kode HTML sederhana dan menggunakan CSS untuk mengatur desain seperti warna, ukuran teks, dan layout. Materi ini memperluas wawasan saya di bidang teknologi dan membuka minat untuk mendalami pengembangan web lebih lanjut.
